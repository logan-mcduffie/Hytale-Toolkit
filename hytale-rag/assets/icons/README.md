# Icon Assets

Place PNG icons (recommended 64x64 or larger) in this folder.

## Required Icons

### IDE Cards (Step 1)
- `vscode.png` - VS Code logo
- `jetbrains.png` - JetBrains logo
- `terminal.png` - Terminal/CLI icon

### Provider Cards (Step 2)
- `copilot.png` - GitHub Copilot logo
- `claude.png` - Anthropic Claude logo
- `gemini.png` - Google Gemini logo
- `codex.png` - OpenAI Codex logo

Icons will be scaled to fit (32x32 for IDE cards, 24x24 for provider cards).
If an icon file is missing, a fallback emoji will be shown instead.
